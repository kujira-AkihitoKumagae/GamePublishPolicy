プライバシーポリシー

本プライバシーポリシーは、本アプリにおけるユーザー情報の取り扱いについて説明するものです。

1. 取得する情報について

本アプリでは、ユーザーの個人情報（氏名、住所、メールアドレス等）を取得することはありません。

2. データの保存について

本アプリでは、ゲームの進行状況などのデータをユーザーの端末内に保存します。
これらのデータは外部サーバーへ送信されることはありません。

3. 外部サイトへのリンクについて

本アプリには、外部のWebサイトへ遷移する機能が含まれています。
ユーザーがボタン操作を行った場合に、外部企業のWebサイトが表示されることがあります。

遷移先のWebサイトにおける個人情報の取り扱いについては、当該サイトのプライバシーポリシーをご確認ください。

4. 第三者提供について

本アプリは、ユーザーの個人情報を第三者に提供することはありません。

5. アクセス解析ツールについて

本アプリでは、アクセス解析ツール等は使用しておりません。

6. 広告について

本アプリでは、第三者配信の広告サービス（AdMob等）は使用しておりません。
ただし、アプリ内から外部企業のWebサイトへ遷移する機能を提供しています。

7. お問い合わせ

本ポリシーに関するお問い合わせは、以下の連絡先までお願いいたします。

メールアドレス： your-email@example.com

8. 改定について

本ポリシーの内容は、必要に応じて変更されることがあります。
